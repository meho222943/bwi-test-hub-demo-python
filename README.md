# python-unittest-demo

Python + `unittest` demo for the **BWI Test Hub**. Register the clone
URL and run — no Docker needed (runs via DIRECT_PROCESS, needs only
`python3`). `run.py` runs the suite in `tests/` and writes
`report/index.html` + `report/junit.xml`.

Make it fail: change an `assertEqual` in `tests/test_calc.py`.
